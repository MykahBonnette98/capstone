Description
-------------

Feature 1: Create profile/profile page
Once clicking the Create Profile on the Main page, you are taken to the second
page with input fields for First Name, Last Name, and Email. Profile information
then will be saved to a database. Once you click save, 
you will see the Home in the top right corner that takes you back to the Main page.
-function code path capstone/public/profile.js

Feature 2: Login username & password
On the main page in the top right corner is the login inputs for a username and
password. Username and password will be checked, if they are valid or invalided the
user will be alerted.
-function code path capstone/public/index.js

Feature 3: Lotr button 
The LOTR button gives the user a random alert with what LOTR character "they are alike".
Each click gives a random/different alert.
-function code path capstone/server/controller.js

Feature 4: Add wishlist destinations
In the Add Wishlist destinations section there is a table with an an input field to
add wish destinations that the user would like to visit. Once clicked, the destination
will be checked and highlighted green. If wished to be removed, user clicks the highlighted
and it returns to the previous color.

Feature 5: Image slideshow in footer
At the bottom in the footer there is a slideshow with images that display each image
after a certain amount of seconds.
